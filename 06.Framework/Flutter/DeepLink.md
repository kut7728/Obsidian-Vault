# 개요
- 앱의 특정 화면으로 바로 이동시키는 링크 기술
- 특정 링크를 클릭하면 앱이 켜지면서 설정된 화면으로 바로 이동하는 것


# 종류
## scheme link
- 앱에서 직접 만든 스킴^[[[URL, URI, URN]]]
- 설정도 쉽고, 애플-안드로이드 양 플랫폼 모두 지원되는 장점
- 하지만 앱이 없을경우 웹 fallback이 안됨
